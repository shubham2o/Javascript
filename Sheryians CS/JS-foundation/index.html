1. Interpreter - Byte Code - Solves task fast but runs slowly - JavaScript, Python

2. Compiler - Machine Code - Solves task slowly but runs fast - C++, Java

3. Just in Time compiler uses Interpreter to solve task fast and uses normal compiler to run fast.
JavaScript uses V8 for Google Chrome and Node.js. SpiderMonkey for Mozilla Firefox. JavaScriptCore for Apple's Safari.
Python uses PyPy JIT.
Java uses Java Virtual Machine.

4. Window is basically an object of the browser, it is not an object of javascript. Window is a collection of tools, features or methods which the browser offers to javascript, such as alert(), document., console.log(), prompt(), etc.

5. Primitive Data Types - String, Number, BigInt, Boolean, null, undefined, and symbol.

6. Reference Data Types - Array [ ], Objects { }, and function ( ).

7. When using Reference Data Types, if you want to make a clone or a copy of one variable value, then with the help of spread operator [... variableName] you can do this. In short, if you want to convert reference data types into the primitive data types, then use spread operator.

8. Falsy Values - null, NaN, undefined, 0, "", '', document.all, false.

9. Loops means repetition. Types of Loops - for loop, while loop, do-while loop, for-in loop, for-of loop, for-each loop. foreach loop works only on arrays, and is used to loop through each key / value pair in an array.

10. In Javascript, function are reusable block of code that helps us to perform a specific task. ES5 has three types of functions - 1. function statements, 2. function expressions, 3. Anonymous functions. ES6 has only one type - 1. Fat Arrow functions.

11. undefined is a value. It is a value given to a variable when there is no value given to a variable. It is treated as a default value or a garbage value.

12. Not defined is an error.

13. Null is also a value. Null is received when something is not found.

14. When you want to store one or more than one value together then use array.

15. Objects in javascript helps us to keep information or details of the similar kind together. Object in javascript helps us to keep information or details of the similar kind together.

16. Array is equal to many people data. Object is equal to many data of a person.

17. Synchronous Code always run line by line. Asynchronous Code does not run line by line, all the async code starts at the same time and whichever task gets completed first, it will get executed first also.

18. Synchronous Code goes to the main stack and asynchronous code goes to the side stack.
