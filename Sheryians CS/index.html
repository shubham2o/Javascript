Interpreter - Byte Code - Solves task fast but runs slowly - JavaScript, Python

Compiler - Machine Code - Solves task slowly but runs fast - C++, Java

Just in Time compiler uses Interpreter to solve task fast and uses normal compiler to run fast. JavaScript uses V8 for Google Chrome and Node.js. SpiderMonkey for Mozilla Firefox. JavaScriptCore for Apple's Safari. Python uses PyPy JIT. Java uses Java Virtual Machine.

Window is basically an object of the browser.
